STEP 1:
Paste your Google Client ID into `.env.local` after:
GOOGLE_CLIENT_ID=

STEP 2:
Paste your Google Client Secret into `.env.local` after:
GOOGLE_CLIENT_SECRET=

STEP 3:
Save the file.

STEP 4:
Run:
npm run google:env-check-and-ship

STEP 5:
Open:
https://www.bodymindharmony.co.uk/admin/bookings

STEP 6:
Click Google sign-in.

STEP 7:
Copy the refresh token from the callback page.

STEP 8:
Paste it into `.env.local` after:
GOOGLE_REFRESH_TOKEN=

STEP 9:
Run again:
npm run google:env-check-and-ship

Emails (optional): set RESEND_API_KEY and FROM_EMAIL in .env.local for client confirmation emails (see .env.example).
